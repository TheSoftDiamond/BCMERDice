# BCMER Dice

A Emergency Dice inspired mod for Brutal Company. When forcing events, all events apply the next day. All rolls are made on the host which means that the host's config settings for Brutal will determine what gets rolled. This addon should also work with any custom events and external events that are loaded into Brutal Company Minus Extra Reborn.

Special thanks to Pacoito, Xu, Boom, Wesley, Tkronix for assistance in helping me with the Unity side of things.

Rolling the d6 has a few outcomes:
 - 1: allows you to force in an event of your choice (50%), a random event gets selected for you (25%), or nothing occurs (25%).
 - 2: Very Bad/Insane event is forced.
 - 3: Bad event is forced.
 - 4: Neutral or Remove event is forced
 - 5: Good event is forced
 - 6: Very Good/Rare event is forced.
